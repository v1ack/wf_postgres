### Работа с внешней базой данных и Windows Forms
##### Владимир Кирилкин ПИ18-2
Для запуска проекта необходима СУБД Postgres со следующими настройками
- адрес `localhost:5432`
- база `windows_app`
- пользователь `me`
- пароль `hackme`

### Зависимости
- Npgsql
- Npgsql.EntityFrameworkCore.PostgreSQL

### Описание
В программе имеются две таблицы `New` `Category`

На первой вкладке представлен список категорий, с возможностью редактирования

На второй вкладке представлен список новостей, с возможностью редактирования

На тертий вкладке результат операции Join двух таблиц, без возможности редактирования, но с возможностью фильтрации данных

Также на первой вкладке пристутствует возможность заполнить базу данных тестовыми значениями
